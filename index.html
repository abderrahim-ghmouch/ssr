2. Qu'est-ce que le CSR ?

Rendu Côté Client (Client-Side Rendering)

Le rendu côté client (CSR) est une technique moderne de développement web où la responsabilité du rendu de la page est confiée directement au navigateur de l'utilisateur.

Au lieu de recevoir un document HTML complet du serveur, le navigateur reçoit un squelette HTML minimal accompagné d'un volumineux fichier JavaScript.

Une fois téléchargé, le JavaScript s'exécute sur l'appareil, récupérant les données nécessaires via des API et construisant dynamiquement l'interface utilisateur.

3. Qu'est-ce que le SSR ?

Rendu Côté Serveur (Server-Side Rendering)

Le rendu côté serveur (SSR) est la méthode traditionnelle d'affichage du contenu web, récemment modernisée par des frameworks puissants.

  • Génération Complète : Le serveur génère la structure HTML complète.
  • Données Pré-remplies : Les requêtes API sont traitées sur le serveur.
  • Prêt à être vu : Le navigateur reçoit un document complet visible instantanément.
  • Charge Prévisible : Déplace le calcul du client vers le serveur.

6. Avantages du CSR

Expérience Fluide

Transitions instantanées, SPA-like.

Charge Serveur Réduite

Fichiers statiques + API légères.

Navigation Rapide

Mise en cache JS, données seules.

7. Avantages du SSR

Affichage Initial Rapide

HTML visible immédiatement.

SEO Supérieur

Indexation complète par robots.

Indépendant du Matériel

Fonctionne sur anciens devices.

8. Limites du CSR

  • Chargement Initial Lent Écran blanc lors de l'analyse JS.
  • Défis SEO Moteurs n'exécutent pas le JS.
  • Dépendance Client Performance liée au matériel.

9. Limites du SSR

  • Requêtes Serveur Fréquentes Aller-retour complet.
  • Coûts d'Infrastructure Puissance serveur élevée.
  • TTFB plus long Attente requêtes DB.

10. Performance & SEO

Métriques de Performance

FCP : SSR gagne. TTI : CSR brille après chargement. TTFB : CSR plus rapide via CDN.

Le Champ de Bataille du SEO

SSR offre une sortie lisible. Les balises Open Graph nécessitent HTML initial. Google diffère le rendu JS.
